# Obsidian Prism Lab

An immersive multi-page creative studio site with sculptural 3D branding, cinematic motion, gallery imagery, ambient audio, and a contact funnel.

## Pages

- `index.html` - flagship manifesto and featured work
- `about.html` - studio identity and team narrative
- `services.html` - offer architecture and process
- `contact.html` - intake form and direct contact surface

## Local Run

```bash
npm install
npm run dev
```

## Validation

Validated with Forge for build integrity, browser flow, accessibility, Lighthouse quality gates, and visual regression.
